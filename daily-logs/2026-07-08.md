## 学到的知识点

### 1. 反向传播 = 误差倒着走

```
前向：输入 → 加权求和 → sigmoid → 输出 → 误差
反向：误差 → 每层梯度 → 局部梯度 × 上游梯度 → 更新权重
```

### 2. 三步走一遍
```
误差 = 0.78 - 1.0 = -0.22（预测小了）

第一步：过 sigmoid
  sigmoid 梯度 = 输出 × (1 - 输出) = 0.78 × 0.22 = 0.17
  信号变成：-0.22 × 0.17 = -0.037

第二步：分给每个输入
  x₁(0.8) → -0.030
  x₂(0.9) → -0.033  ← "好看"分到最多
  x₃(0.3) → -0.011

第三步：更新权重
  w₂ = 0.9 - 0.01 × (-0.033) = 0.90033
```

### 3. 核心公式
**下游梯度 = 上游梯度 × 局部梯度**

每层只管自己的"局部梯度"，跟上面传下来的信号相乘就行。

### 4. 实战技巧速记
- **权重初始化**：不能全为 0，随机小数
- **ReLU**：正数过，负数截断
- **Adam**：自动调学习率
- **过拟合**：只会背答案 → L2 正则化惩罚
