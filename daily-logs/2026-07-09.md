## 学到的知识点

### 1. 依存结构：谁修饰谁
箭头从核心词（Head）指向依赖词（Dependent）。
```
吃 ──→ 我      （谁吃？）
吃 ──→ 苹果    （吃什么？）
吃 ──→ 爱      （什么动作？）
"吃"是核心，其他都依赖它。
```

### 2. 歧义：同一句话，不同画法
```
"警察用刀杀了人"
理解一：杀 → 刀  （工具关系，刀是凶器）
理解二：人 → 刀  （修饰关系，人随身带刀）
```

### 3. Shift-Reduce 解析：像玩扑克牌
```
栈（已处理） | Buffer（未处理）
三个操作：
  Shift:    拿一张牌进栈
  Left-Arc: 建左弧（栈顶第二个 ← 栈顶）
  Right-Arc:建右弧（栈顶 → 栈顶第二个）
```

### 4. 神经依存解析器
用词向量 + POS 向量 输入三层神经网络，预测下一步该做哪个操作（Shift/Left/Right）。快且准。
