如何推送GitHub

你打开终端，先到这个目录：

&#x20;cd C:\\Users\\TX\\nlp-learning-journal

第一步建立md文件

notepad projects\\2026-07-20-fastenhancer.md

第二步把第一步文件加进暂存区

git add projects/2026-07-20-fastenhancer.md

第三步检查状态（红色-新文件未登记，绿色-打包好了等贴标签，白色-干净的都推上去了）

git status

第四步取名字

git commit -m "2026-07-20: FastEnhancer T ONNX模型跑通 方向4启动"

第五步推送

git push
