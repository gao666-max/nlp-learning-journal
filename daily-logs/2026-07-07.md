## 学到的知识点

### 1. 一个神经元，完整的计算（用数字走了一遍）

```python
x = [0.8, 0.9, 0.3]        # 输入："太好看了" 每个词的分数
w = [0.5, 0.9, 0.1]        # 权重："好看"权重最大
b = 0                       # 偏置

z = 0.8×0.5 + 0.9×0.9 + 0.3×0.1 + 0 = 1.24

sigmoid(1.24) = 1 / (1 + e^(-1.24))
              = 1 / (1 + 0.289)
              = 0.78 → 78% 概率是好评
```

### 2. sigmoid 的直观理解
- 不管输入多大或多小，输出永远在 0~1 之间
- z 很大 → 输出接近 1；z = 0 → 输出 0.5；z 是负数 → 输出接近 0
- 就是一个"压缩器"，把任意数字变成概率

### 3. 权重不是人写的，是学的
```
一开始：随机权重 → 预测巨烂 → 误差大
          ↓
      梯度下降：误差大 → 狠狠调权重
          ↓
      再预测 → 误差小了 → 微调
          ↓
      重复几万次 → 模型自己发现了"好看"应该权重大
```
整个过程没有人告诉模型规则，它从数据里自己学的。
