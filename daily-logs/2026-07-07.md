## 学到的知识点

### 1. 神经元 = 加权求和 + 激活函数
```
输入 → 乘权重 → 加偏置 → 过激活函数 → 输出
```
- 类比：台灯旋钮，越往右拧灯越亮
- sigmoid 函数：输入越大 → 输出越接近 1，输入越小 → 输出越接近 0

### 2. 隐藏层：把简单特征组合成复杂判断
- 单层神经元只能画"直线"分类，处理不了语言的复杂逻辑
- 多层（输入层 → 隐藏层 → 输出层）能组合信息
- 例子："虽然烂番茄评分低，但观众都说好"
  - 第一层认出"烂"、"好"、"但"
  - 隐藏层发现"但"推翻了"烂"，"好"才是重点
  - 输出：正面评价

### 3. 激活函数（ReLU）：没有它就白叠了
- 如果每层都只是线性运算，10 层 = 1 层
- ReLU：x > 0 输出 x，x ≤ 0 输出 0
- → "只留有用的信号，没用的扔掉"

### 4. 三层合起来 = 深度学习的全部地基
BERT、GPT、Transformer 都是神经元的堆叠，只是堆法更聪明。
