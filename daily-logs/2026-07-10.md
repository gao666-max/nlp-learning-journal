## 学到的知识点

### 1. 语言模型 = 猜下一个词
```
输入："我今天想去吃____"
预测：火锅 45%、饭 30%、面 12%、考试 0.01%
```
"考试"概率极低 → 模型从海量数据里学到"吃"后面极少出现"考试"。

### 2. RNN 的核心：隐藏状态（记忆口袋）
```
读"我"     → 口袋：[主语=我]
读"今天"   → 口袋：[我 + 时间=今天]
读"想"     → 口袋：[我 + 今天 + 意愿=想]
读"吃"     → 口袋：[我 + 今天 + 想 + 吃]
```
每一步：**新记忆 = 旧记忆 + 新信息**

### 3. RNN 三个核心优势
- **处理任意长度**：不管句子多长都能跑（不像 N-gram 只能看 3 个词）
- **权重共享**：每一步用同一套权重，学的是"规律"不是"位置"
- **记忆功能**：隐藏状态理论上能保留很远的信息

### 4. 训练相关
- 损失函数：交叉熵（希望正确词的概率越高越好）
- 反向传播：BPTT（随时间反向传播），就是 Day 4 的反向传播加上时间维度
- 评估指标：困惑度（越低越好）

### 5. RNN 应用
- 词性标注 / NER
- 情感分析（取最后一个隐藏状态做分类）
- 机器翻译 / 语音识别

## 关键收获
从"处理一个词"正式进入"处理一串词"的阶段。RNN 的隐藏状态是后面 LSTM、Transformer 的基石。
