## 核心概念

### 1. 神经元（Neuron）
深度学习的"原子"——每个神经元做一件事：

```
输入 → 乘权重(W) → 求和 → 加偏置(b) → 过激活函数 → 输出
```

类比：台灯旋钮，输入越大输出越强。

### 2. 隐藏层（Hidden Layer）
- **为什么需要**：单层神经元只能画"直线"分割，语言问题是"曲线"的
- **作用**：把简单特征逐层组合成复杂判断
- 例子："虽然烂，但好看" → 第一层认出词 → 隐藏层发现"但"推翻了"烂" → 输出正面

### 3. 激活函数（Activation Function）
- **为什么必须是非线性的**：线性叠线性还是线性，10 层 = 1 层，白叠了
- **ReLU**：x > 0 输出 x，x ≤ 0 输出 0
  - 本质：丢弃负信号，只留正信号通过

## 关键理解

```
词向量 → 神经元层1 → ReLU → 神经元层2 → ReLU → ... → 最终输出
          ↑                    ↑
    提取简单特征          组合成复杂模式
```

神经元、隐藏层、激活函数三者构成了所有深度学习模型（BERT、GPT、Transformer）的地基。后面的课程只是在这上面换不同的"堆法"。
