## 一个神经元，完整走一遍

**场景**：判断"太好看了"是不是好评。

### 第一步：加权求和

```
输入:  "太"=0.8  "好看"=0.9  "了"=0.3
权重:  ×0.5     ×0.9        ×0.1        ← "好看"权重最大

z = 0.8×0.5 + 0.9×0.9 + 0.3×0.1 + 0 = 1.24
```

### 第二步：sigmoid（把数字变概率）

```
sigmoid(1.24) = 1 / (1 + e^(-1.24))
              = 1 / (1 + 0.289)
              = 0.78

→ 78% 概率是好评 ✓
```

sigmoid 的直观理解：**一个压缩器，输入再大也压到 1 以下，输入再小也压到 0 以上。**

```
z = +∞  → sigmoid ≈ 1.00   "确定是好评"
z =  0  → sigmoid = 0.50   "不确定"
z = -∞  → sigmoid ≈ 0.00   "确定是差评"
```

### 第三步：权重怎么来的？

**权重不是人规定的，是学出来的：**

```
随机权重 → 预测 → 看误差 → 梯度下降调权重 → 再预测
                                              ↓
                                    重复几万次，误差越来越小
                                    模型自己发现了"好看"权重要大
```

## 为什么需要隐藏层？

单层神经元只能画直线分割问题。语言是曲线边界。

```
"虽然烂，但观众都说好"

单层：看到"烂" → 差评（太粗暴了）
多层：第一层识别"烂"、"好"、"但"
      → 隐藏层发现"但"推翻了"烂"，"好"才是重点
      → 输出：好评 ✓
```

---

## 为什么必须加激活函数（ReLU）？

线性叠线性还是线性 → 10 层 = 1 层，白叠了。

**ReLU**：x > 0 输出 x，x ≤ 0 输出 0 → "只留有用信号，没用的扔掉。"

加上非线性，多层才能组合出复杂判断。
