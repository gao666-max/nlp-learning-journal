## 核心概念

### 1. 语言模型（Language Model）
核心任务：给定前文，预测下一个词的概率分布。

```
P(火锅 | "我今天想去吃") = 0.45
P(饭   | "我今天想去吃") = 0.30
P(考试 | "我今天想去吃") = 0.0001  ← 几乎不可能
```

### 2. N-gram 的问题
- 只看前面 n-1 个词（通常 3 个），丢失长距离上下文
- 稀疏性问题：没见过的词组概率为 0
- 存储爆炸：n 增大后组合数量指数增长
- 生成文本逻辑不连贯

### 3. RNN：循环神经网络

```
核心思想：隐藏状态像"记忆口袋"，一边读一边积累。

每一步：h_t = tanh(W·x_t + U·h_{t-1} + b)
                         ↑           ↑
                     当前输入      前一步记忆

新记忆 = 当前信息 + 历史记忆
```

**三个核心优势**：
- 处理任意长度输入（不固定窗口）
- 权重共享（每一步用同一组 W, U，学的是规律不是位置）
- 记忆功能（隐藏状态一直往下传）

### 4. RNN 的训练

**BPTT（随时间反向传播）**：
- 就是 Day 4 的反向传播 + 时间维度
- 误差沿时间回传，各时间步梯度累加

**困惑度（Perplexity）**：
- 衡量语言模型好坏的指标
- 越低越好，本质 = e^(交叉熵)

### 5. RNN 的应用

| 任务类型 | 示例 |
|----------|------|
| 序列标注 | 词性标注、NER |
| 序列分类 | 情感分析（取最后 hidden state） |
| 序列生成 | 机器翻译、语音识别、文本生成 |

## 与前几讲的联系

```
L1-L2: 词向量 → 一个词的表示
L3-L4: 神经网络 + 反向传播 → 怎么算、怎么学
L5: 依存解析 → 多个词的结构关系
L6: RNN → 一串词的序列建模
                 ↓
     L7-L8: LSTM + Transformer（升级版序列模型）
```
