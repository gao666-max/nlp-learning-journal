## 核心概念

### 1. 两种语法结构观
- **短语结构**：词组成嵌套短语（NP, VP 等），类似上下文无关语法
- **依存结构**：词与词之间的修饰关系，箭头从核心词 → 依赖词

### 2. 依存结构示例
```
"我爱吃苹果"
吃 ──→ 我      (谁吃？→ 我，主语关系)
吃 ──→ 苹果    (吃什么？→ 苹果，宾语关系)
吃 ──→ 爱      (什么动作？→ 爱)
```

### 3. 歧义问题
```
"警察用刀杀了人"
解释一：杀 → 刀  （刀是工具，修饰"杀"）
解释二：人 → 刀  （刀是人带的，修饰"人"）
```
计算机需要通过模型判断哪种结构更合理。

### 4. Transition-based 解析：三个操作

```
栈（Stack）     |  Buffer
已处理的词       |  还没处理的词
```

| 操作 | 作用 |
|------|------|
| **Shift** | 把 Buffer 第一个词移入栈顶 |
| **Left-Arc** | 建左弧：栈顶第二个 ← 栈顶，栈顶出栈 |
| **Right-Arc** | 建右弧：栈顶 → 栈顶第二个，第二个出栈 |

### 5. 神经依存解析器
- 输入：词向量 + POS 词性向量
- 模型：三层神经网络
- 预测：下一步该 Shift / Left-Arc / Right-Arc？
- 优势：快（每秒数百句），不需要手工设计特征

### 6. 评估指标
- **UAS**：箭头方向对不对（无标签）
- **LAS**：方向 + 标签（主语/宾语等）对不对
