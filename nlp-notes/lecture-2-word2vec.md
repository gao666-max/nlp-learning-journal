## 核心概念

### 1. 分布式语义假设
一个词的含义由它经常出现的上下文决定。"观其伴而知其义。"

### 2. CBOW vs Skip-gram
- **CBOW**：用周围词预测中心词（多个输入 → 一个输出）
- **Skip-gram**：用中心词预测周围词（一个输入 → 多个输出）
- Word2Vec 主要用的 Skip-gram

### 3. Softmax 函数
把词向量之间的点积（相似度分数）映射为 0~1 的概率分布，所有概率和为 1。
- 问题：全量 Softmax 需要遍历整个词汇表计算，太慢
- 解决：**负采样（Negative Sampling）**——只挑几个"错词"和一个"对词"，计算量从几十万降到几十

### 4. 训练过程
- **目标函数**：衡量预测好坏的数学指标
- **训练**：通过梯度下降不断降低目标函数的值
- **SGD（随机梯度下降）**：每次随机抽一小批数据更新参数，不用等全部数据

## 关键关系
```
词向量 → 点积 → Softmax → 概率 → 对比真实答案 → 损失函数 → 梯度下降 → 更新词向量
                                                                              ↓
                                                                       循环往复，越来越好
```

## 梯度推导深入理解（2026-07-06 补充）

### 为什么 Softmax "低效"（Inefficient）
Softmax 分母有一个 Σ（对全部 V 个词求和）。每预测一个词，计算机要遍历整个词汇表（比如 50 万个词）算一遍。实际应用慢得无法接受。

### 梯度公式的直观含义
推导结果是：**u_o - Σ P(x|c) u_x**

- **u_o（Observed / 观察到的）**：真实的上下文词向量
- **Σ P(x|c) u_x（Expected / 期望的）**：模型根据当前水平猜出来的加权平均

→ **梯度 = 真实 - 预测 = 差值（Difference）**

预测准 → 差值小 → 少调整。预测离谱 → 差值大 → 大调整。神经网络不是靠背诵，是靠不断计算"差值"来学习的。

### 推导中的两个关键数学技巧

**1. 取对数（Log）**
把除法变减法：log(A/B) = log A - log B。方便求导。

**2. 链式法则（Chain Rule）**
Softmax 是嵌套函数，先对外层求导，再对内层求导。
- 第一部分（log A）抵消后剩下真实词向量
- 第二部分（log B）求导后变成所有可能词的加权平均

### 一句话记住
**模型通过"减法"找到错误，通过"梯度"修正错误。** 不需要手写完整推导，但要理解这个逻辑。
