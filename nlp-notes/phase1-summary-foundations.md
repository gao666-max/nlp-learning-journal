## 1. 词向量长什么样？

- **不是 one-hot**：不再是 50 万维里只有 1 个 1 其余全是 0
- **低维稠密**：50~300 维，每个位置都有实数
- **分布式语义**：词的含义"抹"在整个向量空间中，每个维度都参与表达语义

```
phone = [0.456, 0.158, 0.985, -0.363, 0.452, ...]
```

## 2. 相似度怎么算？

- **点积**：两向量相乘求和，越大越相似
- **余弦相似度**：只看角度不看长度，更常用
- **空间邻近性**：意思相近的词自动聚在一起

```
phone 最近的词：text(0.899), screen(0.894), line(0.878)
apple 最近的词：banana(0.999)
```

## 3. 语义运算怎么做？

- 词向量空间不仅捕捉相似性，还捕捉逻辑关系
- **类比推理**：king - man + woman ≈ queen ✅ 已代码验证
- **语法关系**：Japan - Tokyo + China ≈ Beijing ✅ 已代码验证
